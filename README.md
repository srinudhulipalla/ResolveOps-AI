# 🛠️ ResolveOps AI

An intelligent IT Service Desk Copilot powered by Agentic AI. 

ResolveOps acts as a Tier-1 support agent that can autonomously answer policy questions, check system inventories, and file support tickets by coordinating multiple specialized AI agents.

## 🏗️ Architecture
- **Framework:** Google Agent Development Kit (ADK)
- **Backend:** Python / FastAPI
- **Frontend:** Vanilla HTML/CSS/JavaScript
- **Data Layer:** Local SQLite (Inventory/Tickets) + Document RAG (IT Policies)
- **Deployment:** Hugging Face Spaces (Docker)
